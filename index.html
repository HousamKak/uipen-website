02 // Architecture

One script tag.
One MCP channel.

The overlay injects into your dev page via a shadow-DOM script. The server is a single Bun process that binds :8787 and wires stdio to Claude Code via MCP. No extensions. No cloud. No accounts.

01 // BROWSER

Annotate

Shadow-DOM overlay captures the annotation, extracts target metadata, POSTs it.

POST /annotations
<selector, file:line, computed>
02 // SERVER

Channel

Bun + SQLite + MCP stdio. Broadcasts to Claude as a notifications/claude/channel.

notify → <channel source="uipen">
03 // CLAUDE

Edit

Edits the file at the right line. Calls resolve_annotation with the files it touched — so revert knows what to undo.

resolve(id, files_changed)
