Building a dev-mode overlay that pipes pins, regions, and arrows straight into Claude Code via the Channels protocol. Instead of describing a UI bug in prose, you draw on it — and the agent gets the exact element, source file, and design-token values.

What's in the first release:

- Pin, region, and arrow annotations
- Design-token-aware knobs — edit `rounded-xl`, `p-6`, color tokens inline
- `Ctrl+click` multi-select, `Alt+scroll` to tweak values
- `Shift+H` hides the entire panel for clean screenshots
- One-command install: `uipen init`
- Native installers for Windows and Linux (macOS after)
- License keys, 3 activations per seat

Pricing is $29 one-time. No subscription, no cloud, no account. Future major versions are a $14 upgrade every ~18 months — you keep your current version forever.
