Swapped the original map-pin mark for a fountain-pen nib with a lime precision dot at the tip. The old mark read as "location" — wrong for a UI annotation tool.

The nib is now the canonical brand across:

- `uipen.dev` favicon
- Windows installer (`.ico`)
- Linux AppImage desktop entry
- Polar product page
- GitHub avatar (monochrome outline version)

Source SVGs, ICO, and PNG exports live in the [UIpen repo](https://github.com/HousamKak/uipen) under `brand/`. One `bun run build:icons` regenerates every raster size from the single `logo-card.svg` source — that way CI, installers, and this site never diverge.
